자바스크립트 선언문

선언문은 자바스크립트 코드를 작성할 영역을 선언하는 것이라고 이해하면 됩니다.

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Document</title>    
「   <script>                                     
         자바스크립트 코드;         〈--선언문
     </script>                」
</head>
<body>
</body>
</html>

자바스크립트 주석처리

선언문 안에서 코드가 아닌 설명글을 넣고 싶을 때는 주석처리를 하면 됩니다.
이때 한 줄 주석일 경우 '//내용'으로 작성하고 여러 줄 주석일 경우에는 /*내용*/으로 작성하면 됩니다.

//한줄 주석처리

/*
여러 줄 
주석 처리
*/

<!-- HTML 주석 처리 -->

코드 입력시 주의사항

  • 대, 소문자를 구별합니다.
    • 날씨 객체 생성: New date(); (X)
날씨 객체 생성: New Date(); (O)

  • 실행문 뒤에는 세미콜론 = ( ; )을 붙여야 합니다.
    • document.wrute("bye") (X)
document.wrute("bye"); (O)

  • 문자형 데이터 작성시 큰따옴표 = (" ") 와 작은따옴표 = ('') 겹침 오류에 주의해야합니다.
    • document.wrute
("그녀가"안녕"이라고 말했다."); (X)
('그녀가 "안녕"이라고 말했다.'); (O)
("그녀가\"안녕\"이라고 말했다."); (O)

  • 실행문 작성시 괄호의 짝이 맞아야합니다.
    • document.wrute("hi"; (X)
document.write("bye"); (O)

  • 실행문 작성시 한 줄에 한 문장만 작성하는 것이 좋습니다.
