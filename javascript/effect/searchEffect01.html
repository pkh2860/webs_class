indexOf()를 이용하여 CSS 속성 검색하기

CSS 속성 검색하기

  • all : all 속성은 CSS 속성을 재설정하는데 사용할 수 있는 약식 속성입니다.
  • animation : animation 속성은 애니메이션 속성을 설정하기 위한 약식 속성입니다.
  • animation-delay : animation-delay 속성은 애니메이션 시작되는 시간을 설정합니다.
  • animation-direction : animation-direction animation-direction 속성은 움직이는 방향을 설정합니다.
  • animation-duration : animation-duration animation-duration 속성은 움직이는 시간을 설정합니다.
  • animation-fill-mode : animation-fill-mode 속성은 애니메이션이 끝난 후의 상태를 설정합니다.
  • animation-iteration-count : animation-iteration-count 속성은 애니메이션이 반복 횟수를 설정합니다.
  • animation-name : animation-name 속성은 애니메이션 키프레임 이름을 설정합니다.
  • animation-play-state : animation-play-state 속성은 애니메이션의 진행상태를 설정합니다.
  • animation-timing-functionanimation-timing-function : 속성은 애니메이션 객체의 시간 경과에 따른 속도를 정의하는 타이밍 함수를 설정합니다.
  • appearanceappearance : 속성은 기본 양식 컨트롤이 렌더링되는 방식을 제어합니다.
  • backdrop-filter : backdrop-filter 속성은 요소 뒤에 필터 효과를 설정합니다.
  • backdrop-visibility : backdrop-visibility 속성은 요소 뒷면 출력 여부 설정합니다.
  • background: background 속성은 배경 속성을 스타일 시트의 같은 위치에 설정하기 위한 약식 속성입니다.
  • background-attachment : background-attachment 속성은 요소 또는 해당 콘텐츠와 함께 스크롤이되는지 여부를 설정합니다.
  • background-blend-mode: background-blend-mode 속성은 요소의 각 배경 레이어에 대한 혼합 모드를 설정합니다.
  • background-clip : background-clip 속성은 배경이 칠해지는 영역을 지정합니다.
  • background-color : background-color 속성은 요소의 배경색을 설정합니다.
  • background-image: background-image 속성은 요소의 배경 이미지를 설정합니다.
  • background-origin : background-origin 속성은 배경이 칠해질때 배경 이미지의 원점이 어디인지 설정합니다.
  • background-position : background-position 속성은 해당 배경 위치 지정 영역 내에서 크기 조정 후의 초기 위치를 정의합니다.
  • background-repeat : background-repeat 속성은 이미지를 반복할지 여부와 방법을 설정합니다.
  • caption-side : caption-side 속성은 테이블 caption의 위치를 설정합니다.
