HTML 기초 문법

HTML은 Hyper Text Markup Language 약어로 웹 브라우저를 통해
사용자에게 보이는 웹 문서를 작성하기 위헌 언어 입니다.
다시 말해 보여지는 브라우저의 구조를 설계하며 웹 피이지를 위한 마크업 언어입니다.

<!DOCTYPE html>
<html>
    <head>
    <title;>기본 HTML 문법</title>
    </head>
    <body>
        Hello,world!!
    </body>   
</html>
