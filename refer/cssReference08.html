HTML 태그를 통해서 표현하는 방법

Img 태그

이미지의 의미가 있을 때 img태그를 사용합니다.
Alt 태그를 이용해서 대체 문자를 적어야 합니다.


    <img src="이미지 주소 기입" alt="대체 문자 기입">

CSS 속성을 통해서 표현하는 방법

Background 속성

이미지가 의미가 없을 때 background 속성을 사용합니다.
의미가 없기 때문에 대체 문자를 적을 필요가 없습니다.
