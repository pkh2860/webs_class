ID

CSS에서의 정의 방법은 #id name{속성명:속성값; 속성명:속성값;}이런식으로 적용합니다.
id 사용할 때 중요한것은 class와 다르게 한페이지에서 한번만 사용이 가능합니다 id는 페이지 내에서 특정 위치나 태그를 지정하는 것이기 때문에 오직 페이지내에서 한번만 사용할 수 있습니다.

class

자주 사용되는 스타일에 적용하는 것이라고 생각하시면 됩니다 .
예를 들어서 빨강색or파란색 등 색상을 미리 지정하고 html에 적용시키는 방법입니다. css에서 정의하는 방법은
.class name{속성명:속성값; 속성명:속성값;}이런식으로 적용합니다
