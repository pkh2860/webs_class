속성 설명
grid grid 속성은 그리드 레이아웃 속성을 설정합니다.
grid-area grid-area 속성에 다음 유형을 사용하여 아이템을 배치시킬 수 있습니다.
grid-auto-columns grid-auto-columns 속성은 암시적인 그리드 행/열/트랙 크기를 자동으로 설정합니다.
grid-auto-flow grid-auto-flow 속성은 그리드에 명시적으로 배치되지 않은 아이템이 있을 경우 자동 배치 알고리즘이 실행되어 자동으로 배치 되도록 설정합니다.
grid-auto-rows grid-auto-rows 속성은 암시적인 그리드/행/열 트랙 크기를 자동으로 설정합니다
grid-column grid-column 속성은 공백으로 구분된 값 리스트를 해석하여 그리드 행,열을 설정합니다 각 값은 트랙 크기를 말합니다.
grid-column-end grid-column-end 속성은 특정 item을 표시하기 끝마칠 열을 지정합니다.
grid-column-gap grid-column-gap 속성은 열 사이의 간격을 정합니다
grid-column-start grid-column-start 속성은 특정 item을 표시하기 시작할 열을 지정합니다.
grid-gap grid-gap 속성은 grid-row-gap와 grid-column-gap 속성을 같이 사용합니다.
grid-row grid-row 속성은 grid-row-start와 grid-row-end 속성을 같이 사용합니다.
grid-row-end grid-row-end 속성은 특정 item을 표시하기 끝마칠 행을 지정합니다.
grid-row-gap grid-row-gap 속성은 행 사이의 간격을 정합니다.
grid-row-start grid-row-start 속성은 행의 시작 라인을 따라 그리드항목을 라인 기반으로 배치하는 데 사용합니다.
grid-template grid-template 속성은 컬럼의 속성을 설정합니다.
grid-template-areas grid-template-areas 속성은 아이템의 위치를 설정합니다.
grid-template-columns grid-template-columns 속성은 가로 컬럼의 크기와 위치를 설정합니다.
grid-template-rows grid-template-rows 속성은 그리드 행의 크기 조정 기능을 설정합니다.
