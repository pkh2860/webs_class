식별자/키워드

식별자

식별자란 변수나 함수의 이름이 될 수 있는 프로그래머가 선언하는 단어이다.

식별자의 규칙

1. 식별자는 최소한 하나의 문자로 구성되어야 한다.
2. 식별자를 구성하는 첫 번째 글자는 반드시 하나의 [문자]나 , [ _ ], [ $ ]로 시작되어야 한다.
3.첫 글자 다음에 오는 각 글자는 [문자], [ _ ], [ $ ], [숫자]여야 한다.
4.[ _ ]와 [$]를 제외한 특수문자와 공백은 식별자를 구성할 수 없다.

식별자 이름을 구성하는 방법

식별자의 이름을 구성하는 방법에 관해 그 어떤 자바스크립트 법칙도 관여하지 않고 있지만
자바스크립트 커뮤니티에서는 몇 가지 비공식적인 표준을 가지고 있다.
그 중 하나가 낙타표기법(CamelCase)이다


낙타표기법이란 식별자 이름이 한 단어 이상으로 구성될 경우,
대소문자를 혼합하여 사용하는 방법이다.


num_cake_donuts 변수 식별자 이름에 _ 를 사용해서 열러 개의 단어를 분리하는 방법
NumCakeDonuts 위의 방법이 낙타표기법이다.
각 단어의 첫 번째 글자가 대문자이다.
numCakeDonuts 이번 방법은 소문자 낙타표기법이다.
첫 번째 단어를 제외한 각 단어의 첫 글자가 대문자로 구성되있다.

변수는 보통 소문자 낙타표기법을 사용하는데 이것은 첫 번째 단어는 모두 소문자이지만
나머지 단어들은 대문자를 혼합하는 표기법이다

키워드

키워드란 자바 스크립트가 처음 만들어질 때 정해진, 특별한 의미가 있는 단어이다.
모든 브라우저에서 아래와 같은 키워드들을 지원하며,이 키워드들은 식별자로 정의할 수 없다.

키워드의 종류

break loop문이나 switch문을 종료하기 위해 사용.
continue loop문을 빠져나가 가장상단으로 가는데 사용.
debugger 자바스크립트의 실행을 중지하고 (가능하다면)디버깅 function을 부른다.
do ... while do while반복문 실행
function function을 선언한다
lf ... else if문 실행
return function에서 나온다.
switch switch문 실행
try ... catch error를 핸들링하는 try catch문 실행
var 변수를 선언한다

키워드의 규칙

자바스크립트 키워드는 예약어로서, 변수명으로 사용할 수 없다.
